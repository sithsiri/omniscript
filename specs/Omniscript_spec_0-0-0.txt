Omniscript Specification 0.0.0

Key:
[]   type of input
()   optional type of input
/    or
*!*  comment

Place text on the screen:
echo [text]
print [text]

Set a variable:
set [variable]=[string/value/variable](arithmetic expression)(second value/variable)
[variable]==[string/value/variable]
[variable]=[string/value/variable]
